#define TM_READLINE ( stream_in,
string_out,

Read a line from a stream into a String; non-blocking. Reads in a single line (terminated by \r or \n or \r\n) Processes backspaces. Optionally echoes chars back to the input stream.

Note that this routine uses the TM_YIELD macro. As such, it needs to be supplied with a yield_id value to use in the TM_YIELD. Also, any routine using this will need to be a TaskManager task and use one of the TM_BEGIN family of macros along with the accompanying TM_END.

Parameters
stream_in– the input stream. Also the output stream if echoing.
string_out– a String object. NOT cleared at entry.
yield_id– a unique integer to be used for the contained TM_YIELD
echo– true if chars are to be echoed back to the input stream.
